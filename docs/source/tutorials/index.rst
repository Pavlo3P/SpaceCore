Tutorials
=========

These pages are the Sphinx versions of the tutorial notebooks. The first
tutorials explain the core abstractions before the API reference: backend
execution, contexts, spaces, linear operators, and conversion policy.

The :doc:`regularized optimal transport tutorial <regularized_ot>` is an
application example. It keeps the executed cell outputs in the rendered docs so
the convergence diagnostics and transport-plan plots can be inspected directly.
The notebook is also available
`on GitHub <https://github.com/Pavlo3P/SpaceCore/blob/main/tutorials/6_Regularized_Opt_Transport.ipynb>`_.
